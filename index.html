Custom Light Board (Arduino Uno R3)

Summary
  • This project is a custom light board powered/controlled by an Arduino Uno!

  • The wooden platform was built to fit a 3 x 3 grid (each section 4" x 4") under frosted glass (3/32" thick). In each section of the grid, there is an individual LED; each LED is connected to one another through wiring underneath the board.

  • The board is controlled by the user through a button on the side. Whenever the button is pressed, the color palette running through the LEDs changes (5 possible light patterns).


Challenges
  • Soldering the individual LEDs...This task came to be very tedious because for each LED there are 6 connections to be soldered. However, it was a good opportunity to improve my soldering skills more & more.

  • Woodworking...While I am an aspiring Electrical/Computer Engineer, the woodworking may have been my favorite part of this project :) But, holding the platform sides to the base when trying to screw/glue them together became an issue because I did not have enough clamps! So, this process took longer than it should have.

Reactive LED Cube (Arduino Uno R3)

1 / 4
2 / 4
3 / 4
4 / 4
❮ ❯
Summary
  • This project is a reactive LED cube powered/controlled by an Arduino Uno and a modified IR proximity sensor!

  • The cube was constructed as a 4" x 4" x 3" wooden box. The cube holds an IR proximity sensor and an individual LED, covered on top by glass (clear, 3/32" thick).

  • Users may interact with this device by placing items on its surface. Whenever something is placed on top of the glass, the LED lights up green; otherwise, the LED remains blue.


Challenges
  • Modified IR Sensor...In order to allow the IR transmitter and receiver to properly work through glass (at this distance), the transmitter was disconnected and soldered onto longer wire connections. This allowed the angle of the signal to bounce properly through the glass. This aspect of the sensor was very finicky and difficult to calibrate.

  • Woodworking...I did not have access to power tools for this project, so hand sawing wooden pieces of this size was difficult!

Finalized December 2021

Summary
  • This project is an arcade gaming console powered by a Raspberry Pi!

  • The LCD screen was salvaged from an old monitor, and the frame was built to fit the screen's dimensions; the screen is held in place with metal backings (tabs) and aluminum tape. The electrical components/wiring are housed directly under the screen. An old mouse and keyboard are used as the HIDs.

  • Users can access Pacman, Tetris, Snake, and Space Invaders through the command line. Furthermore, I uploaded a self-developed Python script Pong game that users can run through a Python IDE.
(Someone assisted me in decorating the final product)


Challenges
  • Building the frame...This part was difficult because I did not have access to any power tools. So, the wooden frame was cut using a hand saw, which made it very difficult and tedious to cut out the screen container. Still, however, it was a lot of fun :)

  • The Raspberry Pi...The Raspberry Pi I used was something I received at an IT camp in 10th grade and is a model that was released in 2014. So, it was quite slow to actually get it running. However, it had no issue running simple games. Great way to put it to use!

Presence Detecting LED Strip (Arduino Uno R3)

1 / 3
2 / 3
3 / 3
❮ ❯
Summary
  • The result of this project is a presence activated LED strip that is installed in my room!

  • This project is powered by an Arduino Uno and uses a Passive IR motion sensor (PIR motion sensor) to detect any motion from a warm body (from humans). Whenever a presence is detected, the Arduino runs a red/purple color pattern for ~60 seconds.

  • I incorporated a photoresistor so the device is only activated when the room is dim enough (at night when lights are off, dimmer lighting, etc.)


Challenges
  • Circuit Wiring/Organizing...I accidentally ordered the wrong wires, so (at first) I was unable to have most of the wires flush against the breadboard. Instead, I had to use longer jumper wires which made the device look like a big jumble of cables. I felt like I had to fix it, but when I got new solid-core electrical wires, I made a mistake putting my circuit back together. It took a while to figure out where I messed up and turned out to be a frustrating process, oops!

Finalized October 2021

Summary
(This is NOT a personal project but I still wanted to include it!)

  • This is a project I completed in my Circuits and Signals: Biomedical Applications course! I find this particular project very cool because I have experience working in the medical field (as an EMT) and have used commercial ECG devices many times.

  • The circuit created incorporated instrumentation amplifiers and high/low filters for signal processing. Through these methods, and gain steps, I was able to achieve a clean ECG signal.

  • Through 3 electrode leads, I could test this circuit on my own body and see my own heart rhythym!


Challenges
  • Signal processing...When processing the signal from my own body, I was encountering a lot of unusual noise. I thought it was something wrong with my circuit or calculations for a while, but I discovered that the issue was my desk's setup! Having my laptop charging and/or having a monitor connected was somehow causing this problem. It was a relief it wasn't an issue with my calculations! :)

Finalized April 2021
